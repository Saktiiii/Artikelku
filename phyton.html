Dasar Pemrograman Python

üìÖ 1 Oktober 2024 ‚è±Ô∏è 10 menit baca üë§ Admin RPL

Python adalah salah satu bahasa pemrograman yang paling populer dan mudah dipelajari di dunia. Bahasa ini diciptakan oleh Guido van Rossum dan pertama kali dirilis pada tahun 1991. Python terkenal dengan sintaksnya yang sederhana dan mudah dibaca, membuatnya menjadi pilihan ideal untuk pemula yang ingin belajar pemrograman.

üéØ Mengapa Memilih Python?

Python memiliki banyak keunggulan yang membuatnya menjadi bahasa pemrograman yang sangat populer:

üí° Keunggulan Python:

  • Sintaks Sederhana: Python menggunakan sintaks yang mirip dengan bahasa Inggris
  • Cross-Platform: Dapat berjalan di Windows, macOS, dan Linux
  • Library Lengkap: Memiliki ribuan library untuk berbagai keperluan
  • Komunitas Besar: Dukungan komunitas yang sangat aktif
  • Versatile: Dapat digunakan untuk web, AI, data science, dan automasi

üöÄ Memulai dengan Python

Instalasi Python

Sebelum mulai coding, kita perlu menginstall Python terlebih dahulu. Kunjungi situs resmi Python di python.org dan download versi terbaru. Setelah instalasi selesai, buka command prompt dan ketik:

python --version

Jika instalasi berhasil, akan muncul versi Python yang terinstall.

Program Python Pertama

Mari kita buat program Python pertama kita - program "Hello World" yang klasik:

# Program Hello World dalam Python
print("Hello, World!")
print("Selamat datang di dunia Python!")

üìù Konsep Dasar Python

1. Variabel dan Tipe Data

Python memiliki berbagai tipe data dasar yang perlu kita pahami:

# Variabel dan tipe data
nama = "Budi"           # String
umur = 17               # Integer
tinggi = 170.5          # Float
aktif = True            # Boolean

print(f"Nama: {nama}")
print(f"Umur: {umur} tahun")
print(f"Tinggi: {tinggi} cm")
print(f"Status aktif: {aktif}")

2. Operasi Matematika

Python mendukung berbagai operasi matematika dasar:

# Operasi matematika
a = 10
b = 3

print(f"Penjumlahan: {a} + {b} = {a + b}")
print(f"Pengurangan: {a} - {b} = {a - b}")
print(f"Perkalian: {a} * {b} = {a * b}")
print(f"Pembagian: {a} / {b} = {a / b}")
print(f"Pembagian bulat: {a} // {b} = {a // b}")
print(f"Sisa bagi: {a} % {b} = {a % b}")
print(f"Pangkat: {a} ** {b} = {a ** b}")

3. Input dari User

Kita dapat mengambil input dari pengguna menggunakan fungsi input():

# Input dari user
nama = input("Masukkan nama Anda: ")
umur = int(input("Masukkan umur Anda: "))

print(f"Halo {nama}, Anda berumur {umur} tahun")

# Menghitung tahun lahir
tahun_sekarang = 2024
tahun_lahir = tahun_sekarang - umur
print(f"Anda lahir pada tahun {tahun_lahir}")

üèãÔ∏è Latihan 1: Kalkulator Sederhana

Buatlah program kalkulator sederhana yang dapat melakukan operasi penjumlahan, pengurangan, perkalian, dan pembagian. Program harus meminta input dua angka dari user dan menampilkan hasil semua operasi.

üí° Lihat Solusi 
# Kalkulator Sederhana
print("=== KALKULATOR SEDERHANA ===")
angka1 = float(input("Masukkan angka pertama: "))
angka2 = float(input("Masukkan angka kedua: "))

print(f"\nHasil operasi:")
print(f"{angka1} + {angka2} = {angka1 + angka2}")
print(f"{angka1} - {angka2} = {angka1 - angka2}")
print(f"{angka1} * {angka2} = {angka1 * angka2}")

if angka2 != 0:
    print(f"{angka1} / {angka2} = {angka1 / angka2}")
else:
    print("Tidak dapat membagi dengan nol!")

üîÑ Struktur Kontrol

Conditional (If-Else)

Struktur kontrol memungkinkan program membuat keputusan berdasarkan kondisi tertentu:

# Contoh penggunaan if-else
nilai = int(input("Masukkan nilai ujian: "))

if nilai >= 90:
    grade = "A"
    keterangan = "Excellent!"
elif nilai >= 80:
    grade = "B"
    keterangan = "Good job!"
elif nilai >= 70:
    grade = "C"
    keterangan = "Not bad"
elif nilai >= 60:
    grade = "D"
    keterangan = "Need improvement"
else:
    grade = "F"
    keterangan = "Study harder!"

print(f"Nilai: {nilai}")
print(f"Grade: {grade}")
print(f"Keterangan: {keterangan}")

Perulangan (Loops)

Python memiliki dua jenis loop utama: for dan while.

# For loop
print("Counting dengan for loop:")
for i in range(1, 6):
    print(f"Hitungan ke-{i}")

print("\nTabel perkalian 5:")
for i in range(1, 11):
    hasil = 5 * i
    print(f"5 x {i} = {hasil}")

# While loop
print("\nCountdown dengan while loop:")
countdown = 5
while countdown > 0:
    print(f"Countdown: {countdown}")
    countdown -= 1
print("Blast off! üöÄ")

üìö Tips Belajar Python

üéØ Tips Sukses Belajar Python:

  • Praktek Setiap Hari: Luangkan minimal 30 menit setiap hari untuk coding
