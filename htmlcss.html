Materi Html dan CSS

üìÖ 1 Oktober 2024 ‚è±Ô∏è 10 menit baca üë§ Admin RPL

HTML (HyperText Markup Language) dan CSS (Cascading Style Sheets) adalah fondasi dari setiap website. HTML digunakan untuk menyusun struktur halaman, sedangkan CSS digunakan untuk memperindah tampilannya.

üìÑ HTML: Struktur Halaman

HTML terdiri dari berbagai tag seperti <h1>, <p>, <a>, dan lain-lain untuk menyusun elemen halaman.

<!DOCTYPE html>
          <html>
            <head>
              <title>Contoh HTML</title>
            </head>
            <body>
              <h1>Halo Dunia!</h1>
              <p>Ini adalah paragraf pertama.</p>
            </body>
          </html>

üé® CSS: Desain Tampilan

CSS digunakan untuk mengatur warna, ukuran, layout, dan responsivitas halaman web.

body {
            background-color: #f4f4f4;
            color: #333;
            font-family: Arial, sans-serif;
          }
          
          h1 {
            color: #0d47a1;
          }

üîó Menggabungkan HTML dan CSS

CSS bisa disisipkan dalam HTML menggunakan tag <style> atau disimpan di file eksternal.

<!DOCTYPE html>
          <html>
            <head>
              <style>
                h1 {
                  color: green;
                }
              </style>
            </head>
            <body>
              <h1>Judul Berwarna Hijau</h1>
            </body>
          </html>

üìå Kesimpulan

Dengan menguasai HTML dan CSS, kamu dapat membangun halaman web yang terstruktur rapi dan menarik secara visual. Selanjutnya kamu bisa mempelajari JavaScript untuk membuat halaman yang interaktif.
