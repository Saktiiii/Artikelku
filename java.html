Instalasi Java Development Kit (JDK)

Sebelum menulis program Java, kita perlu menginstall JDK terlebih dahulu:

üì• Langkah Instalasi:

  1. Download JDK dari oracle.com atau openjdk.java.net
  2. Install JDK sesuai dengan sistem operasi Anda
  3. Set environment variable JAVA_HOME
  4. Verifikasi instalasi dengan membuka terminal/command prompt
# Verifikasi instalasi Java
java -version
javac -version

Program Java Pertama

Mari kita buat program Java pertama - program "Hello World":

// File: HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        System.out.println("Selamat datang di dunia Java!");
    }
}

Untuk menjalankan program Java:

# Compile program
javac HelloWorld.java

# Jalankan program
java HelloWorld

2. Input dari User

Untuk mengambil input dari user, kita menggunakan class Scanner:

import java.util.Scanner;

public class InputDemo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("Masukkan nama Anda: ");
        String nama = scanner.nextLine();
        
        System.out.print("Masukkan umur Anda: ");
        int umur = scanner.nextInt();
        
        System.out.print("Masukkan tinggi Anda (cm): ");
        double tinggi = scanner.nextDouble();
        
        // Output hasil
        System.out.println("\n=== PROFIL SISWA ===");
        System.out.println("Nama: " + nama);
        System.out.println("Umur: " + umur + " tahun");
        System.out.println("Tinggi: " + tinggi + " cm");
        
        // Menghitung tahun lahir
        int tahunSekarang = 2024;
        int tahunLahir = tahunSekarang - umur;
        System.out.println("Tahun lahir: " + tahunLahir);
        
        scanner.close();
    }
}

3. Operasi Matematika

Java mendukung semua operasi matematika dasar:

public class OperasiMatematika {
    public static void main(String[] args) {
        int a = 15;
        int b = 4;
        
        System.out.println("=== OPERASI MATEMATIKA ===");
        System.out.println("a = " + a + ", b = " + b);

        System.out.println("Penjumlahan: " + a + " + " + b + " = " + (a + b));
        System.out.println("Pengurangan: " + a + " - " + b + " = " + (a - b));
        System.out.println("Perkalian: " + a + " * " + b + " = " + (a * b));
        System.out.println("Pembagian: " + a + " / " + b + " = " + (a / b));
        System.out.println("Sisa bagi: " + a + " % " + b + " = " + (a % b));
        
        // Operasi dengan Math class
        System.out.println("\n=== OPERASI MATH CLASS ===");
        System.out.println("Pangkat: " + Math.pow(a, 2));
        System.out.println("Akar kuadrat: " + Math.sqrt(a));
        System.out.println("Nilai maksimum: " + Math.max(a, b));
        System.out.println("Nilai minimum: " + Math.min(a, b));
    }
}

üèãÔ∏è Latihan 1: Kalkulator BMI

Buatlah program untuk menghitung Body Mass Index (BMI). Program harus meminta input berat badan (kg) dan tinggi badan (m), kemudian menghitung BMI dan memberikan kategori kesehatan.

üí° Lihat Solusi 
import java.util.Scanner;

public class KalkulatorBMI {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        System.out.println("=== KALKULATOR BMI ===");
        System.out.print("Masukkan berat badan (kg): ");
        double beratBadan = input.nextDouble();
        
        System.out.print("Masukkan tinggi badan (m): ");
        double tinggiBadan = input.nextDouble();
        
        // Hitung BMI
        double bmi = beratBadan / (tinggiBadan * tinggiBadan);
        
        // Tentukan kategori
        String kategori;
        if (bmi < 18.5) {
            kategori = "Kurus";
        } else if (bmi < 25.0) {
            kategori = "Normal";
        } else if (bmi < 30.0) {
            kategori = "Gemuk";
        } else {
            kategori = "Obesitas";
        }
        
        System.out.println("\n=== HASIL ===");
        System.out.printf("BMI Anda: %.2f\n", bmi);
        System.out.println("Kategori: " + kategori);
        
        input.close();
    }
}

Conditional Statements

Java menggunakan if-else untuk membuat keputusan berdasarkan kondisi:

import java.util.Scanner;

public class SistemNilai {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        
        System.out.print("Masukkan nilai ujian (0-100): ");
        int nilai = input.nextInt();
        
        String grade;
        String keterangan;
        
        if (nilai >= 90) {
            grade = "A";
            keterangan = "Excellent! Sangat baik!";
        } else if (nilai >= 80) {
            grade = "B";
            keterangan = "Good! Baik sekali!";
        } else if (nilai >= 70) {
            grade = "C";
            keterangan = "Fair. Cukup baik.";
        } else if (nilai >= 60) {
            grade = "D";
            keterangan = "Poor. Perlu belajar lebih giat.";
        } else {
            grade = "F";
            keterangan = "Fail. Harus mengulang!";
        }
        
        System.out.println("\n=== HASIL EVALUASI ===");
        System.out.println("Nilai: " + nilai);
        System.out.println("Grade: " + grade);
        System.out.println("Keterangan: " + keterangan);
        
        input.close();
    }
}

Perulangan (Loops)

Java memiliki tiga jenis loop: for, while, dan do-while:

public class PerulanganDemo {
    public static void main(String[] args) {
        // For Loop
        System.out.println("=== FOR LOOP ===");
        System.out.println("Tabel perkalian 7:");
        for (int i = 1; i <= 10; i++) {
            System.out.println("7 x " + i + " = " + (7 * i));
        }
        
        // While Loop
        System.out.println("\n=== WHILE LOOP ===");
        System.out.println("Countdown:");
        int countdown = 5;
        while (countdown > 0) {
            System.out.println("Countdown: " + countdown);
            countdown--;
        }
        System.out.println("Blast off! üöÄ");
        
        // Do-While Loop
        System.out.println("\n=== DO-WHILE LOOP ===");
        System.out.println("Bilangan genap 1-10:");
        int num = 2;
        do {
            System.out.print(num + " ");
            num += 2;
        } while (num <= 10);
        System.out.println();
        
        // Enhanced For Loop (For-Each)
        System.out.println("\n=== ENHANCED FOR LOOP ===");
        String[] buah = {"Apel", "Jeruk", "Mangga", "Pisang"};
        System.out.println("Daftar buah:");
        for (String item : buah) {
            System.out.println("- " + item);
        }
    }
}

üèóÔ∏è Object-Oriented Programming

Java adalah bahasa yang sepenuhnya berorientasi objek. Mari kita buat class sederhana:

// File: Siswa.java
public class Siswa {
    // Atribut (Fields)
    private String nama;
    private int umur;
    private String jurusan;
    private double nilaiRataRata;
    
    // Constructor
    public Siswa(String nama, int umur, String jurusan) {
        this.nama = nama;
        this.umur = umur;
        this.jurusan = jurusan;
        this.nilaiRataRata = 0.0;
    }
    
    // Getter methods
    public String getNama() {
        return nama;
    }
    
    public int getUmur() {
        return umur;
    }
    
    public String getJurusan() {
        return jurusan;
    }
    
    public double getNilaiRataRata() {
        return nilaiRataRata;
    }
    
    // Setter methods
    public void setNilaiRataRata(double nilai) {
        if (nilai >= 0 && nilai <= 100) {
            this.nilaiRataRata = nilai;
        }
    }
    
    // Method untuk menampilkan info
    public void tampilkanInfo() {
        System.out.println("=== INFO SISWA ===");
        System.out.println("Nama: " + nama);
        System.out.println("Umur: " + umur + " tahun");
        System.out.println("Jurusan: " + jurusan);
        System.out.println("Nilai Rata-rata: " + nilaiRataRata);
        System.out.println("Status: " + getStatus());
    }
    
    // Method untuk menentukan status kelulusan
    public String getStatus() {
        if (nilaiRataRata >= 75) {
            return "LULUS";
        } else {
            return "BELUM LULUS";
        }
    }
}
