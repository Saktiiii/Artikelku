Pemrograman Web Dinamis dengan PHP

üìÖ 22 September 2024 ‚è±Ô∏è 13 menit baca üë§ Admin RPL

PHP (PHP: Hypertext Preprocessor) adalah bahasa pemrograman server-side yang sangat populer untuk pengembangan web dinamis. Dikembangkan pertama kali oleh Rasmus Lerdorf pada tahun 1994, PHP telah menjadi salah satu bahasa pemrograman web yang paling banyak digunakan di dunia, powering lebih dari 70% website di internet termasuk Facebook, WordPress, dan Wikipedia.

üåê Mengapa PHP untuk Web Development?

PHP memiliki banyak keunggulan yang membuatnya ideal untuk pengembangan web:

üéØ Keunggulan PHP:

  • Server-Side Processing: Kode dijalankan di server, hasil dikirim ke browser
  • Open Source & Free: Gratis dan memiliki komunitas besar
  • Cross-Platform: Berjalan di Windows, Linux, macOS
  • Database Friendly: Mudah integrasi dengan MySQL, PostgreSQL, Oracle
  • Easy to Learn: Sintaks mirip C dan Perl, mudah dipelajari
  • Rich Library: Ribuan fungsi built-in dan framework

üîß Persiapan Lingkungan PHP

Instalasi XAMPP/WAMP/LAMP

Untuk development PHP, kita butuh server lokal. Pilihan terpopuler:

üñ•Ô∏è Paket Server Lokal:

  • XAMPP: Cross-platform (Windows, Mac, Linux) - Apache, MySQL, PHP, Perl
  • WAMP: Windows - Apache, MySQL, PHP
  • LAMP: Linux - Apache, MySQL, PHP
  • MAMP: Mac - Apache, MySQL, PHP

Setelah instalasi, verifikasi PHP dengan command:

php -v

üöÄ Program PHP Pertama

Mari kita buat file PHP pertama. Buat file hello.php di folder htdocs (XAMPP):

<?php
// File: hello.php
echo "<h1>Hello World dari PHP!</h1>";
echo "<p>Selamat datang di dunia web development dengan PHP</p>";

// Menampilkan tanggal dan waktu
echo "<p>Sekarang: " . date('Y-m-d H:i:s') . "</p>";

// Informasi PHP
echo "<p>Versi PHP: " . phpversion() . "</p>";
?>

Buka browser dan akses: http://localhost/hello.php

1. Variabel dan Tipe Data

PHP adalah bahasa loosely typed, variabel dimulai dengan tanda $:

<?php
// Variabel PHP dimulai dengan $
$nama = "Budi Santoso";        // String
$umur = 17;                    // Integer
$tinggi = 170.5;               // Float
$aktif = true;                 // Boolean
$hobi = ["coding", "gaming", "reading"];  // Array

// Output
echo "<h2>Data Siswa RPL</h2>";
echo "<p>Nama: " . $nama . "</p>";
echo "<p>Umur: " . $umur . " tahun</p>";
echo "<p>Tinggi: " . $tinggi . " cm</p>";
echo "<p>Status: " . ($aktif ? "Aktif" : "Tidak Aktif") . "</p>";

// Menampilkan array
echo "<p>Hobi: ";
foreach ($hobi as $h) {
    echo $h . " ";
}
echo "</p>";

// Cek tipe data
echo "<p>Tipe data nama: " . gettype($nama) . "</p>";
echo "<p>Tipe data umur: " . gettype($umur) . "</p>";
?>

3. Array dan Fungsi

PHP memiliki array yang sangat fleksibel dan fungsi yang powerful:

<?php
// Array indexed
$bahasa = ["PHP", "JavaScript", "Python", "Java", "C++"];

// Array associative
$siswa = [
    "nama" => "Andi Pratama",
    "kelas" => "XI RPL 1",
    "nilai" => [
        "PHP" => 85,
        "JavaScript" => 90,
        "Database" => 88
    ]
];

echo "<h2>üìä Data Siswa dan Nilai</h2>";

echo "<h3>Profil Siswa:</h3>";
echo "<p>Nama: " . $siswa["nama"] . "</p>";
echo "<p>Kelas: " . $siswa["kelas"] . "</p>";

// Menampilkan nilai
echo "<h3>Nilai Mata Pelajaran:</h3>";
$total_nilai = 0;
$jumlah_mapel = count($siswa["nilai"]);

foreach ($siswa["nilai"] as $mapel => $nilai) {
    echo "<p>$mapel: $nilai</p>";
    $total_nilai += $nilai;
}

$rata_rata = $total_nilai / $jumlah_mapel;
echo "<p><strong>Rata-rata: " . number_format($rata_rata, 2) . "</strong></p>";

// Fungsi untuk menentukan grade
function hitungGrade($nilai) {
    if ($nilai >= 90) return "A";
    elseif ($nilai >= 80) return "B";
    elseif ($nilai >= 70) return "C";
    elseif ($nilai >= 60) return "D";
    else return "F";
}

echo "<h3>Grade Mata Pelajaran:</h3>";
foreach ($siswa["nilai"] as $mapel => $nilai) {
    $grade = hitungGrade($nilai);
    echo "<p>$mapel: $grade</p>";
}

// Array methods yang berguna
echo "<h3>Bahasa Pemrograman Populer:</h3>";
echo "<p>Jumlah bahasa: " . count($bahasa) . "</p>";
echo "<p>Bahasa pertama: " . $bahasa[0] . "</p>";
echo "<p>Bahasa terakhir: " . end($bahasa) . "</p>";

// Mengurutkan array
sort($bahasa);
echo "<p>Urutan alfabetis: " . implode(", ", $bahasa) . "</p>";
?>
